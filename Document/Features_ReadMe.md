# English Vocab Master - 功能說明文件 (README)

本文件概括 English Vocab Master 的三大核心面板及其功能。

## 📱 三大核心面板

### Panel 1: 智能圖片辨識 (AI OCR Extract)
本面板提供 **雙作業模式**，可在頂部切換選單中自如切換「單張圖片辨識」與「資料夾批次處理 📂」：

- **單張圖片辨識模式**：
    - **匯入圖片**：支援點擊或拖曳上傳與貼上手機截圖。
    - **AI 深度分析**：
        - **分群提取**：自動識別圖片中的前綴/群組單字（例如：圖片中有 under-, underwater, underground），AI 會將其拆分為多個獨立單字條目進行學習。
        - **結構化數據**：精確提取單字、音標 (IPA)、繁體中文翻譯與 1-3 句實用例句。
    - **即時預覽與標記**：在存入雲端前，可點擊喇叭圖示聆聽發音，或預先將難記單字標記星號。
    - **存入雲端與自動清理**：
        - 一鍵同步至 Firebase 雲端單字本。
        - **智能清理與 Chrome 專屬刪除**：存入雲端後自動清空預覽，若為 Chrome 瀏覽器，更會詢問是否刪除原始截圖檔案。

- **資料夾批次處理模式 (Folder Batch Processing)**：
    - **原生資料夾選取**：藉由 Chrome 的 File System Access API，讓使用者一鍵選擇本機電腦或手機的題目截圖資料夾（讀取時需授與彈出視窗的「檢視與編輯」權限）。
    - **自動過濾與掃描**：瞬間讀取內含的所有 PNG / JPG / WEBP 格式圖片，並以批次佇列列表優雅呈現。
    - **AI 批次對照雲端資料庫**：
        - 系統逐張解碼圖片並透過 AI 運算提煉內藏單字。
        - **對照比對**：與使用者本人的雲端資料庫交叉比對。若單字已被收錄在雲端單字書中，系統將**以最新圖片提取的情境挑戰（題目）100% 覆蓋覆寫 (Overwrite) 雲端對應單字的題目**，確保以最新教育提示為最優優先級！
    - **無感本機磁碟釋放 (Auto Clean-up 🗑️)**：
        - 若辨識及與雲端對照存儲成功，系統就會透過原生 Chrome API，**自動直接從您的本機裝置資料夾中刪除該圖片檔案**，達到邊辨識邊清空資料夾的極致智慧管理体验！
        - *技術標準依據請參見：[Chrome 官方 File System Access 規範](https://developer.chrome.com/docs/capabilities/web-apis/file-system-access#deleting_files_and_folders_in_a_directory)*

### Panel 2: 雲端單字書 (Library Management)
- **單字清單**：直覺式列出所有已收藏的單字與學習進度（按字母 A~Z 排序）。
- **一體化高效率篩選列 (Unified Filter Ribbon)**：將原先繁雜的兩排式分類面板（常忘與測驗狀態）精煉為**全新單排一體化快捷操作欄**，僅保留最純粹的高頻按鈕：
    - **全部**：一鍵返回完整名冊。
    - **常忘單字 ⭐**：迅速定位所有標記星號、亟需鞏固的頑固生詞。
    - **已存 AI 測驗 🪄**：聚焦已生成專屬 AI 挑戰情境與題目、可隨時開測的項目。
    - **待生成測驗 📝**：精準標明尚未配置測驗內容之生詞，提供動態字數統計，引領智慧化學習。
- **美學極致折行卡片 (Aesthetic Zero-Truncation Layout)**：
    - 徹底捨棄傳統 `truncate`（省略號 `...`）的死板截斷，解決長片語、整句例句或豐富中文解釋在不同螢幕寬度下因遮蔽而無法閱讀的痛點。
    - 引進頂部自適應對齊 (`items-start`) 佈局，利用 `break-words whitespace-normal` 動態物理折行，確保字元、音標及說明不論長短，皆能以優雅呼吸感、多行結構完美呈現。
- **單字連播 (Sequential Playback)**：核心功能。支援全清單自動循環播放，包含自動發音、自動滾動選中單字，並提供暫停鍵，方便用戶在通勤或睡前進行沈浸式單字記憶。
- **詳細查看**：點擊卡片可展開查看完整的翻譯、音標與例句。
- **即時發音**：隨時隨地練習單字發音。
- **清單管理**：可手動刪除已精通的單字。
- **常忘單字 (Hard to Remember)**：點擊星號即可標記。
- **動態統計**：清單上方顯示「單字總數」，會隨著搜尋篩選或切換「全部/常忘」標籤實時更新數量。

### Panel 3: 智能拼寫測驗 (AI Adaptive Quiz)
- **隨機出題**：從資料庫中自動挑選單字進行拼寫挑戰。
- **雙模作答**：
    - **鍵盤模式**：標準手動輸入。
    - **語音模式**：透過麥克風口說測驗，並自動支援片語空格辨識。
- **反饋與診斷系統**：
    - **精準字母比對診斷 (Spelling Guide)**：逐字元分析使用者輸入的答案，透過綠色正確、紅色刪除線直觀指出拼錯點。
    - **自清除與拼字留存**：回答出錯時，系統會自動在輸入框保留使用者「拼對的字母」，而將拼錯/缺失的字母取代為底線 (下劃線 `_`)，避免重新手打的挫折，提供極佳的互動重試體感。

## 🎯 本日重大更新重點 (2026/06/08)

### 1. 雲端儲存防禦 (Undefined Field Prevention)
- **原因**：修正因 Google Gemini 模型回傳 JSON 物件常有部分選填屬性（如 `comment`）在前端序列化時保持為 `undefined`，進而引發 Firestore 寫入崩潰的儲存失敗錯誤。
- **作法**：在所有寫入/更新 Firestore 的端點（QuizPanel、ImportPanel 等）加入深度清理過濾機制與預防性 mapping，100% 保證沒有任何物件欄位為 `undefined`，實現永久穩定存檔。

### 2. 圖片優先覆寫原則 (Image-First Precedence / Multiple Override)
- **規則**：當批次處理截圖並將單字匯入雲端時，若該單字已被收錄且已存有別的情境題目，系統將**以圖片提取的情境內容為最高優先 100% 覆寫 (Override) 舊題目**，解決新截圖題目無法覆蓋資料庫舊教材的問題。

### 3. 微秒級拼字留存交互 (Smart Input Retention)
- **優勢**：在 Quiz 答錯時，自動轉換成「保留正確字、其餘變底線」的智慧重試機制，不干擾主輸入框操作，又提供精準的字母對齊指導。

### 4. 測驗體驗流暢度升級：音標去噪與延遲語音公布 (Quiz UX Upgrades: No-Chaff Phonetics & Delayed TTS Memory reinforcement)
- **問題排除**：排除因部分裝置不支援特殊國際音標編碼，導致在測驗卡下方顯示代表亂碼的「空白方框 (Rectangle Boxes)」問題，改為完全拿掉此音標提示，保持卡片介面簡約純淨。
- **作答沉浸感優化**：預設在作答未完成前不顯露例句發音按鈕，避免提前劇透。當使用者**確認答對或公布解答**的那一霎那，播放朗讀按鈕將配合優雅的 bounce 動畫彈跳現身，引導使用者點擊整句朗讀發音，透過「答完後聆聽」加深單字在真實英文例句中的聲學語境印象。
