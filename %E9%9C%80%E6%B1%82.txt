# VocabMaster AI 需求文件

## 核心功能清單

### Panel 1: 智能圖片擷取 (AI OCR & Study)
- **輸入方式**：支援拖曳或點擊上傳手機螢幕截圖。
- **AI 處理 (Gemini API)**：
    - 從圖片中自動辨識英文單字與文法重點。
    - 生成對應的中文翻譯。
    - 生成國際音標 (IPA)。
- **互動功能**：
    - **發音按鈕**：點擊後以標準美式/英式發音讀出單字。
    - **存檔按鈕**：將單字、音標、翻譯存入雲端資料庫 (Firebase Firestore)。

### Panel 2: 雲端單字本 (Library Management)
- **數據展示**：列出所有已儲存的單字清單。
- **管理動作**：
    - 查看單字詳情。
    - **刪除按鈕**：將該單字從雲端資料庫中移除。

### Panel 3: AI 智慧測驗 (Adaptive Quiz)
- **題目生成**：從雲端單字本隨機挑選單字進行測驗。
- **題型**：填空題。
- **回答方式**：
    - **手動輸入**：鍵盤輸入答案。
    - **麥克風輸入**：語音辨識 (STT) 轉為文字作答。
- **提示系統 (Hints)**：
    - 顯示字母總數。
    - 顯示部分字母 (如首尾字母)。
    - 錯誤反饋 (標示打錯的特定字母)。

## 技術棧
- **Frontend**: React + Tailwind CSS (UI) + Framer Motion (動畫)
- **AI**: Google Gemini 1.5 Flash (OCR + 語言分析)
- **Backend**: Firebase Firestore (雲端儲存)
- **Browser APIs**: Web Speech API (發音與語音辨識)
